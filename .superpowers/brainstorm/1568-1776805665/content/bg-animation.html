Animación de fondo

El BgCanvas actual tiene partículas flotantes simples. ¿Cómo lo evolucionamos?

Neural Network Mesh

A — Neural Mesh Conectado

Nodos que se conectan con líneas cuando se acercan, como un grafo de red neuronal. Evolución directa del actual — más inteligente y relevante para software/AI. Reemplaza partículas simples.

Grid + Pulse

B — Grid Técnico con Pulsos

Una cuadrícula de puntos fina (dot grid) con ondas de pulso que se expanden periódicamente — como un radar o circuito PCB activo. Muy "engineering", sutil y elegante.

Ambos

C — Neural Mesh + Grid (capas)

Combinación: grid de puntos estático como base, con nodos que se mueven y conectan encima. Más profundidad visual. Funciona especialmente bien en el Hero y secciones clave.

Las 3 animaciones están corriendo en vivo. Haz clic para seleccionar.
